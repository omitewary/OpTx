# OpTx

## Instructions

### Clone the Repository
```bash
git clone https://github.com/omitewary/OpTx.git
```
## Install Dependencie
```bash
cd OpTx
npm install
```

## Generate Triangle
### Run following command
```bash
node task1.js
```
### Enter M and N values on prompt

### Output
Displays MxN Triangle

## List products in json
### Run following command
```bash
node task2.js
```
### Output
Creates a products.json file
